## Default Avatar

Use this example to create a circle and rounded avatar on an image element.

```html
<img
  class="w-10 h-10 rounded-full"
  src="/docs/images/people/profile-picture-5.jpg"
  alt="Rounded avatar"
/>
<img
  class="w-10 h-10 rounded"
  src="/docs/images/people/profile-picture-5.jpg"
  alt="Default avatar"
/>
```
